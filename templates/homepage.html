CROPSENSE

This project is based on scientific research demonstrating that multispectral and thermal imagery, combined with decision-tree classification, can effectively detect plant stress and disease. Building on these findings, this platform represents the transition from research to a practical, user-oriented solution.

CropSense provides a centralized web application where users can access project documentation, explore research results, and run the full processing pipeline. The system is designed to support image acquisition, storage, and automated analysis, enabling the detection of plant diseases within a single, intuitive interface.
